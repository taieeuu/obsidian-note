這是Flyte的核心組件之一。它包含使用 protobuf 訊息的 Flyte 語言規範、gRPC 中的後端 API 規範和 Swagger REST。該儲存庫包含產生的多種語言的客戶端和協定訊息結構。除了產生的程式碼之外，儲存庫還包含 Flyte 後端 API 的 Golang 用戶端（在 FlyteAdmin 下分組的服務）。
